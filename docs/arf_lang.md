# The Arf Functional Programming Language

![arflang_icon](icon.png)

This document is an informal specification/proposal for a new functional
programming language with different aspects.

## Guiding principles
Arf is a functional programming language which solves the complexity of understanding 
other functional programming languages and maximum performance with a high-level approach.

## The design is motivated by the following guidelines:
- Functional
- Strongly typed
- A new memory management approch

## Notation
The syntax is specified using Extended Backur-Naur Form (EBNF). In particular:

- `''` encloses lexical symbols
- `|` seperate alternatives
- `()` used for grouping
- `[]` specifies option (0 or 1 times)
- `{}` specifies repetition (0 to n times)
